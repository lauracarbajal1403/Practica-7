## Descripción
- Aplicación web desarrollada con React y Vite que consume una API de
películas y muestra un listado y el detalle de cada película.
## Objetivo
- Desarrollar una aplicación web en React usando Vite como entorno de desarrollo, que consuma
información de una API de películas, muestre un listado y el detalle de cada película, y que además sea
trabajada colaborativamente mediante GitHub y publicada en Netlify.

## Instalación y ejecución local
- npm install / pnpm install -> Instalar todas las instancias y bibliotecas necesarias para el sitio web
- npm run dev / pnpm dev -> Comandos para crear un servidor local y ejecutar el sitio web
- npm build / pnpm build -> Comando para traducir el código y dejarlo preparado para subir a internet
- (Nota: Si inicia el sitio usando comandos npm, usar solo npm de manera exclusiva para evitar conflictos, así mismo, si se inicia con comandos pnpm, solo utilizar pnpm).
## Rutas
- /.github/workflows -> Despliegue en Netlify al hacer push
- /src -> Todo el flujo del sitio web
- /src/components -> Las tarjetas de películas y footer
- /src/data -> Conexión con la API
- /src/pages -> La página que muestra todas las peliculas y los detalles individuales de cada pelicula
- /src/routers -> Enrutamiento de React
- /src/utils -> Imagen placeholder en caso de que no cargue la imagen de la pelicula
