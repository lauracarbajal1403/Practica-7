## Descripción
Aplicación web desarrollada con React y Vite que consume una API de
películas y muestra un listado y el detalle de cada película.
## Tecnologías
- React
- Vite
- React Router DOM
- CSS
- GitHub
- Netlify
## Instalación
npm install
npm run dev
## Rutas
- /.github/workflows -> despliega en netlify al hacer push
- /src -> /components -> Las tarjetas de películas
- /src -> data -> conexión a la api
- /src -> pages -> landing
